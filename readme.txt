Set description and keywords metadata for your website.

== Description ==

Swahili Seo is a plugin developed to assist website owners to easily set seo metadata for their published pages or posts.  

Swahili Seo will be fully supported and maintained until  as long as is necessary.

At a glance, this plugin adds the following:

* Administrators can now set description metadata for post,page or a custom post.
* Administrators can now set keywords metadata for post,page or a custom post.

== Installation ==
1. Upload 'plugin-directory' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' SubPanel in WordPress

== Frequently Asked Questions ==

= Default settings =

 No additional settings are needed, just go to "Add New" for page, post or a custom post then at the bottom you will see Swahili Seo metabox.

=   My Search Engine priview don't update in block Editor (Gutenberg) =

* Refresh the page.

= 	My permalink is different in Search Engine priview  =

* Publish the post/page fisrt to get correct permalink.

= 	This plugin supports Woocommerce  =

* Yes it supports.
